Recipe API is allowed only for authenticated users. Follow this steps to create an account in order to authenticate and use Recipe API:

1. Create user. It's recommended to use mock credentials e.g. jack@example.com

2. Create token and copy value from quotation marks.

4. Click on and perform following steps:.

- Navigate to the top of the website and click on "Authorize" icon.

- Choose tokenAuth (apiKey) from available authorizations and paste your copied value like this:

"Token >your_token<" and click on "Authorize".

Remember that the white space between "Token" and >your_token< is necessary for successful authorization.

If everything was done properly you should see "Authorized" below "tokenAuth (apiKey)".

Now you are allowed to create your own recipes! Go to /api/docs/ and try any endpoint you want :-)
